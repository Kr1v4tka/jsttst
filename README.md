# Physics & Graphics Simulations

Коллекция физических и графических симуляций, реализованных на JavaScript.

## Описание

Проект представляет собой набор интерактивных симуляций, демонстрирующих различные физические явления и графические эффекты. Каждая симуляция работает независимо и может быть запущена в браузере.

## Структура проекта

```
├── README.md           # Документация проекта
├── index.html          # Главная страница со списком симуляций
├── css/                # Стили оформления
├── js/                 # Общий код и утилиты
└── simulations/        # Папки с отдельными симуляциями
    ├── simulation-1/   # Первая симуляция
    ├── simulation-2/   # Вторая симуляция
    └── ...
```

## Запуск

1. Откройте файл `index.html` в любом современном браузере
2. Выберите интересующую симуляцию из списка
3. Взаимодействуйте с симуляцией согласно инструкциям на экране

## Требования

- Современный браузер с поддержкой ES6+
- Включенная поддержка Canvas API (для графических симуляций)

## Разработка

### Добавление новой симуляции

1. Создайте новую папку в директории `simulations/`
2. Реализуйте логику симуляции в отдельном JavaScript-файле
3. Добавьте запись в таблицу доступных симуляций в данном файле
4. Обновите `index.html` при необходимости
