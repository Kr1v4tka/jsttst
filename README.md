# Physics & Graphics Simulations

Интерактивные симуляции на JavaScript.

## Запуск

Откройте `index.html` в браузере.

## Симуляции

- **Песочница с физикой** (`simulations/sandbox/`) — гравитация, столкновения, интерактивное добавление объектов
- **Стерео-глубина** (`demo/`) — вычисление глубины с помощью стереозрения

## Структура

```
├── index.html          # Главная страница
├── Stereo.html         # Демо стерео-глубины
├── css/                # Стили
└── simulations/        # Симуляции
    └── sandbox/        # Физическая песочница
```
